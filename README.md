## 仓库介绍
本仓库用于整理和归档大学期间的学习笔记，包含**理论知识梳理**、**编程实践代码**、**嵌入式开发笔记**三大核心板块，旨在形成结构化、可复用的学习知识库，同时方便复习与知识复盘。

笔记格式以 $\LaTeX$ 排版为主（理论课），搭配 Markdown 笔记（技术实践），兼顾排版美观与内容实用性。

## 维护说明
- 笔记更新：随课程进度和学习计划持续补充，优先完善核心课程与高频使用的技术内容。
- 格式规范：理论课优先使用 $\LaTeX$ 排版，保证公式与排版美观；技术实践优先使用 Markdown，搭配代码块与注释。
- 问题反馈：若发现笔记错误或有补充建议，可通过邮箱 3071974740@qq.com 联系，或提交 Issue。

## 使用说明
- 理论课笔记（如概率论）：可直接下载 $\LaTeX$ 源码编译为 PDF，或查看仓库中已生成的 PDF 文件；
- 技术实践笔记：代码片段可直接复制使用，嵌入式工程代码已适配对应开发板（后续补充具体板型）；
- 建议使用 VS Code + Markdown All in One 插件查看 Markdown 笔记，体验更佳。
